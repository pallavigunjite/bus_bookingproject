# 🚌 Online Bus Ticket Booking System

A simple frontend-based **Bus Ticket Booking Web Application** built using HTML, CSS, and JavaScript. This project allows users to enter travel details and simulate booking a bus ticket.

---

## 🚀 Features

* Enter source and destination
* Select travel date
* Choose number of seats
* Simple login button (demo)
* Instant booking confirmation display
* Clean and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run

1. Download or clone the repository:

```
git clone https://github.com/your-username/bus-booking-app.git
```

2. Open the project folder

3. Run the application:

* Open `index.html` in your browser

---

## 💡 How It Works

* User enters travel details (From, To, Date, Seats)
* Clicks on **Book Ticket**
* JavaScript validates input fields
* Displays booking confirmation on screen

---

## 🔐 Login Functionality

* Basic demo login button
* Shows alert message on click
* No backend authentication (for learning purpose)
